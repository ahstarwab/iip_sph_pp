# IIP speech preprocessing

C library for speech preprocessing.

## Target Platform
* Ubuntu 16.04 x64 (or higher)
* Windows 10 x64
* Mac high sierra 10.13.5
* 임베디드 보드(라즈베리 파이나 zynx 보드 등)

## Requirement
* cmake 3.6 or higher  
* (OPTION) OpenBLAS  
* (OPTION) IntelMKL  
* (OPTION) OpenMP
